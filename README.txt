This archive contains all of the scripts used to perform the analyses detailed in the HiFive paper.

To run this, you can simply set the needed paths in the 'dependencies.txt' file in the 'Settings' folder and then use the command 'make all' in the top-level folder. Please note that this will take a long time.

In order for everything to work, you must have several packages and programs already installed. These include:

Libraries:
hdf5

Python packages:
numpy
scipy
h5py
statsmodels

Programs:
samtools
bowtie
R (2.*)

R packages:
maxLik
Rsge
R.utils
